# 🔑 The Password Game - Mobile App (React Native)

React Native + Expo로 만든 **The Password Game** 스타일 퍼즐 게임입니다.

## 🚀 시작 방법

```bash
npm install
npx expo start
```

## 📂 폴더 구조

```
/components
/screens
/contexts
/utils
```

## 🎮 주요 기능

- 다양한 비밀번호 규칙 적용
- 규칙 실시간 검증
- 레벨 올라가며 규칙 추가
- 애니메이션, 성공 화면 제공

## 🛠️ 기술 스택

- React Native
- Expo
- React Navigation
- Context API 상태 관리
