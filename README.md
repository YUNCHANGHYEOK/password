# Password Game (Universal Version)

이 프로젝트는 Android, iOS, Web에서 실행 가능한 React Native + Expo 기반 게임입니다.

## 실행 방법

### ✅ 웹에서 실행
```bash
npm install
npm run web
```

### 📱 Android APK 빌드
```bash
eas build -p android --local
```

### 🍏 iOS 빌드 (macOS 필요)
```bash
eas build -p ios
```

## 필요 조건
- Node.js & npm
- `eas-cli` 설치: `npm install -g eas-cli`
- Expo CLI 설치: `npm install -g expo-cli`
